# 🔌

A simple (and incomplete) discord lib written in [Emojicode](https://www.emojicode.org/)

Here's a quick `echo` example:

```
📜🔤client/client.🍇🔤

🏁🍇

    🔤Some random token🔤 ➡️ token

    🆕🔌🆕token❗️ ➡️ client

    💞client 🔤message🔤 🍇msg_⚪️
        🍺🔲msg_ 🖊 ➡️ msg
        ↪️🎼💬msg❓ 🔤.echo🔤❗️ 🍇
            🔈client 🏠msg❓ 🔪💬msg❓ 5 📐💬msg❓❗❗️❗️
        🍉
    🍉❗️

    🏃client❗️

🍉
```
